ポジションについて

ポジションはプレイヤーがテーブルに着く位置のことです。


6人でポーカーをした場合の各ポジションの名称は以下の通りになります。




ポジション名 内容
SB ボタンの左側で強制参加費を支払うポジション
BB SBの左側で強制参加費を支払うポジション
UTG プリフロップで、最初にアクションを行うポジション
HJ UTGの次にアクションをするポジション
CO HJの次にアクションをするポジション
BTN COの次にアクションするポジション


ポジションの優劣

ポジションはポーカーをプレーする上で、とても重要です。

ポーカーは、そもそも情報が不十分な中戦うゲームなので、相手から得られる情報はとても貴重なのです。
そのため、相手のアクションを見てから判断できるポジションの方が有利になり、これをインポジション(IP)と呼びます。
逆に、先にアクションをするポジションをアウトオブポジション(OOP)と呼びます。


次に、アクションの順番を見てみましょう。

プリフロップのアクションの順番は、
UTG >> HJ >> CO >> BTN >> SB >> BB

フロップ・ターン・リバーのアクションの順番は、
SB >> BB >> UTG >> HJ >> CO >> BTN


つまり、プリフロップでは、 UTG が1番弱いポジションになります。
フロップ・ターン・リバーでは、 SB が1番弱いポジションとなります。
1番最後にアクションをするのはBTNなので、 BTN が最も強いポジションとなります。




なるべく強いポジションで参加して、悪いポジションではできるだけ参加をしないことが重要になります。


それでは、具体例を見てみましょう。




例えば、あなたはBTNです。
プリフロップで、COからのレイズにコールして、参加しました。フロップが開いたあとに、COがチェックしたとします。


この時点で

CO目線
・BTNはプリフロップでコール

BTN目線
・COはプリフロップでレイズ
・COはフロップでチェック

という情報量の偏りがある状態で、BTNはアクションすることができるのです。
この偏りは、プリフロップからリバーまで続くので、ポジションが後ろであることが物凄く有利であることが分かります。

まとめ

ポジションは、ポーカーをするうえで、アクションの順番が決まるので重要になります。
後でアクションをした方が、相手のアクションを見て、有利に立ち回ることができるので、ポジションが有利な時にゲームに参加しましょう。
